三、已修复的关键 Bug

Bug 1: 空状态巨大空白（用户截图反馈 x3）

根因：height={rows} + flexGrow=1 在空状态下把输入框推到终端最底部

修复：AlternateScreen（深色背景填充）+ justifyContent="flex-end"（内容贴近输入框）+ flexShrink=0（底部固定）

Bug 2: 键盘输入落到终端 shell（用户截图反馈 x2）

根因：useInput({ isActive: false }) 在 prompt_ready 之前不进入 raw mode

修复：isActive: !permissionBlocked（始终捕获 stdin）+ Enter 提交 gated on active prop

Bug 3: React "Cannot update App while rendering InputArea"

根因：setValue wrapper 在 render 期间调用 onChange → parent setInputValue

修复：onChange 改用 useEffect + useRef，在 commit phase 后通知 parent
